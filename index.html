Anatomia de un elemento

Un elemento de HTML consta de una etiqueta de apertura, contenido y una etiqueta de cierre. Por ejemplo:

<h1>Titulo principal</h1>

En este caso, <h1>es la etiqueta de apertura¨Titulo principal¨ es el contenido </h1>

Los atributos proporcionan informacion extra. Ejemplo: class="nota"
