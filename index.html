Anatomia de un elemento

Un elemento de HTML consta de una etiqueta de apertura, contenido y una etiqueta de cierre. Por ejemplo:

<h1>Titulo principal</h1>

En este caso, <h1>es la etiqueta de apertura¨Titulo principal¨ es el contenido </h1>

Los atributos proporcionan informacion extra. Ejemplo: class="nota"

HTML semantico

El HTML semantico utiliza etiquetas que describen, claramente su significado y funcion en su documento.

Ejemplos de etiquetas semanticas

  • <header>:Define el encabezado de una seccion o pagina
  • <nav>:Define un bloquer de navegacion
  • <main>:Define el contenido principal del documento
  • <article>:Define un articulo independiente
  • <section>:Define una seccion generica del documento
  • <footer>:Define el pie de pagina de una seccion.
No uses div para tod, elige la etiqueta adecauada.Recuerda que el HTML SEMANTICO mejora la accesibilidad y SEO.

Contenedores: div vs span

<div> es un contenedor de bloque que se utiliza para agrupàr elementos y aplicar estilos o scripts o un bloque completo. es decirocupa todo el ancho disponible. Por ejemplo:

<div class="caja">

<span>es un contenedor en linea que se utiliza para agrupar elementos de texto y aplicar estilos o sripts es decir no ocupa todo el ancho disponible solo lo que se ocupe del elemento.

<span class="resaltado">Texto resaltado</span>
